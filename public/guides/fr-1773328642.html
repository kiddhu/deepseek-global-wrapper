Exemple Pratique d'Intégration avec l'API SeekAPI

La migration vers SeekAPI.ai est délibérément simple. L'exemple suivant en Python démontre comment configurer le client OpenAI officiel pour pointer vers l'infrastructure de SeekAPI.ai et interagir avec le modèle DeepSeek R1.

Prérequis : Installez la bibliothèque `openai` et obtenez votre clé API sur le tableau de bord SeekAPI.ai.

import openai

# Configuration de l'API pour utiliser SeekAPI.ai
client = openai.OpenAI(
    api_key="votre_clé_api_seekapi_ici",  # Remplacez par votre clé réelle
    base_url="https://dash.seekapi.ai/v1"  # Le point de terminaison SeekAPI
)

def poser_question_a_deepseek(question):
    """Fonction utilitaire pour interroger DeepSeek R1."""
    try:
        response = client.chat.completions.create(
            model="deepseek-r1",  # Spécifiez le modèle DeepSeek R1
            messages=[
                {"role": "system", "content": "Vous êtes un assistant IA utile et précis."},
                {"role": "user", "content": question}
            ],
            temperature=0.7,  # Contrôle le caractère aléatoire des réponses
            max_tokens=500
        )
        return response.choices[0].message.content
    except Exception as e:
        return f"Une erreur est survenue : {e}"

# Exemple d'utilisation
if __name__ == "__main__":
    requete = "Explique le concept de 'reasoning planning' dans les LLMs comme DeepSeek R1."
    reponse = poser_question_a_deepseek(requete)
    print("Question :", requete)
    print("\nRéponse de DeepSeek R1 via SeekAPI.ai :\n")
    print(reponse)

Cet exemple illustre la simplicité de l'intégration. La seule différence notable avec un appel standard à l'API OpenAI est la configuration du `base_url`. Toutes les autres fonctionnalités (streaming, fonctions, etc.) sont supportées de manière compatible.
