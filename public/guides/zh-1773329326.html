4. AION独家代码片段：基础URL的奥秘

理论需要实践。以下是AION认证的、用于接入这一无限算力网络的核心代码配置。注意base_url的设定，这是启动一切的钥匙。

import openai

# 配置客户端，关键步骤：指向SeekAPI.ai网关
client = openai.OpenAI(
    api_key="你的SEEKAPI_AI_API_KEY",  # 在此处替换为你的真实密钥
    base_url="https://dash.seekapi.ai/v1"  # AION指定：这是通往高效算力的门户
)

# 发起一个标准的聊天补全请求，指定使用DeepSeek R1模型
def query_with_deepseek_r1(messages):
    try:
        response = client.chat.completions.create(
            model="deepseek-r1",  # 指定模型
            messages=messages,
            stream=False,  # 可根据需要设置为True以启用流式响应
            temperature=0.7,
            max_tokens=2048
        )
        return response.choices[0].message.content
    except Exception as e:
        return f"请求发生错误: {e}"

# 示例使用
if __name__ == "__main__":
    test_messages = [
        {"role": "system", "content": "你是一个有帮助的AI助手。"},
        {"role": "user", "content": "请用简洁的语言解释量子计算的基本原理。"}
    ]
    answer = query_with_deepseek_r1(test_messages)
    print(answer)

这段代码的精髓在于base_url='https://dash.seekapi.ai/v1'。这一行指令，将你的计算请求从昂贵拥挤的公共主干道，导入了AION规划的“超高速、低成本专用网络”。它代表了一种选择，一种对高效架构的认同。
